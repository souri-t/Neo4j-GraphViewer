# 共有フォルダ権限申請とアクセス不可対応

共有フォルダにアクセスできない場合、フォルダパス、利用者 ID、所属部署、必要な権限、エラーメッセージを確認する。権限不足、グループ反映待ち、ファイルサーバー障害、VPN 未接続、端末キャッシュが主な原因である。

権限追加はフォルダオーナーの承認が必要である。読み取り権限と更新権限を分け、個人単位ではなく権限グループへ追加する。経理、人事、法務のフォルダは機密区分が高いため、所属長とフォルダオーナーの両方の承認を必要とする。

申請が承認されると、ID 管理システムでグループ追加を行う。反映には最大 30 分かかる。反映後もアクセスできない場合は、VPN 接続状態、Windows 資格情報キャッシュ、ファイルサーバーの監査ログを確認する。
