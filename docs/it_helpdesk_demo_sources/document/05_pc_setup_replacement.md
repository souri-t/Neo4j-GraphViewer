# 貸与 PC の初期設定と交換対応

新規貸与 PC は、資産番号登録、OS 更新、EDR 導入、ディスク暗号化、標準アプリ配布、プリンタ設定、VPN クライアント設定、端末証明書発行を行ってから利用者へ引き渡す。初回ログイン時には SSO と MFA 登録を確認する。

PC 故障交換では、旧端末の状態、データ退避、BitLocker 回復キー、ライセンス移行、端末証明書の失効、新端末への証明書発行を確認する。紛失や盗難の場合は通常交換ではなくセキュリティインシデントとして扱い、遠隔ワイプとアカウント保護を優先する。

外出先の利用者へ PC を発送する場合は、初回 VPN 接続に必要な端末証明書と MFA が利用可能であることを事前に確認する。証明書未発行の端末では VPN 接続できないため、社内ネットワークでの事前キッティングが必要である。
