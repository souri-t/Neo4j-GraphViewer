# 社内ITヘルプデスク運用概要

社内ITヘルプデスクは、社員からの問い合わせ、アカウント申請、端末トラブル、ネットワーク障害、セキュリティインシデントを一次受付する。対象システムは SSO、MFA、VPN、Microsoft 365、社内 Wi-Fi、共有フォルダ、主要 SaaS、貸与 PC である。

問い合わせはチケット管理システムで受け付け、カテゴリ、影響範囲、緊急度、利用者部門、端末種別、発生時刻を記録する。ログイン不能、業務停止、セキュリティ疑いは優先度を上げる。単なる操作質問と障害の切り分けを早く行うことが重要である。

よくある問い合わせは、外出先から VPN につながらない、スマートフォン変更後に MFA が使えない、パスワード誤入力でアカウントロックされた、共有フォルダにアクセスできない、退職者のアカウント停止手順を確認したい、である。
