# 退職者の SaaS 権限とライセンス回収

退職時は SSO アカウントを無効化するだけでなく、Microsoft 365 ライセンス、CRM、経費精算、勤怠、電子契約、プロジェクト管理ツールの SaaS アカウントを停止する。SSO 連携 SaaS でも、SaaS 側にローカルアカウントや管理者権限が残る場合がある。

退職者のメール転送、共有フォルダ権限、メーリングリスト、端末証明書、貸与 PC 回収、スマートフォン回収を一覧で確認する。緊急退職や懲戒案件では、退職日を待たずに SSO、VPN、MFA、SaaS 権限を即時停止する。

ライセンス回収はコスト削減だけでなく、情報漏えい防止の観点でも重要である。JML 棚卸しでは、退職者、異動者、休職者について、不要な SaaS ロールと共有フォルダ権限が残っていないか確認する。
