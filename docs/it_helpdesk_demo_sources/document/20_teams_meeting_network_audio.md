# Teams 会議の音声不良とネットワーク確認

Teams 会議で音声が途切れる、映像が止まる、会議に参加できない場合、端末性能、ネットワーク品質、マイクとカメラの権限、Teams クライアント版数、Microsoft 365 サービス正常性を確認する。特定フロアで多発する場合は社内 Wi-Fi やプロキシ、アクセスポイントの混雑を疑う。

単一利用者では、ヘッドセット、Bluetooth 接続、Teams キャッシュ、Outlook 会議リンク、MFA や SSO ログイン状態を確認する。複数利用者が同じ会議に参加できない場合は、会議ポリシー、ライセンス、Microsoft 365 側の障害、ネットワーク制限を確認する。

役員会議や顧客向け説明会など代替不可の会議で影響が出ている場合は、チケット優先度を P1 または P2 に上げる。暫定回避策として、ブラウザ版 Teams、電話会議、別ネットワーク、モバイル回線の利用を案内する。
