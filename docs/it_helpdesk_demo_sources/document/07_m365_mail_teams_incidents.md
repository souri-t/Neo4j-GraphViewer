# Microsoft 365 メール・Teams 障害対応

Outlook や Teams の問い合わせでは、単一利用者の設定問題か、Microsoft 365 側のサービス障害かを切り分ける。単一利用者の場合は、ライセンス、SSO ログイン、MFA、メールボックス容量、クライアントキャッシュ、ネットワークを確認する。

複数部署から「メールが届かない」「Teams 会議に参加できない」と同時に連絡がある場合は、Microsoft 365 管理センターのサービス正常性、社内ネットワーク、DNS、プロキシを確認する。サービス障害が疑われる場合は、既知障害として社内ポータルに掲載し、個別チケットには一括回答する。

Teams には入れるがメールだけ使えない場合、Exchange Online のライセンス、メールボックス容量、Outlook プロファイル破損、転送ルール、迷惑メール判定を確認する。メールは届くが通知されない場合は、Outlook クライアントとモバイルアプリの通知設定を確認する。
