# パスワードリセットとアカウントロック解除

パスワードを 5 回連続で誤入力すると、アカウントは 15 分間ロックされる。急ぎの場合は本人確認後にヘルプデスクがロック解除を行う。パスワードリセットはセルフサービスを原則とし、MFA が利用できない場合のみヘルプデスクが一時パスワードを発行する。

本人確認では、社員番号、所属部署、上長名、登録済み電話番号、直近の出社状況を確認する。メールだけでの依頼はなりすましリスクがあるため、登録済み電話番号への折り返しを推奨する。

ロックが繰り返し発生する場合、古いパスワードを保存したメールアプリ、VPN クライアント、スマートフォンの Wi-Fi 設定、業務アプリの資格情報が原因になることがある。解除だけで終わらせず、どの端末やアプリから失敗ログが出ているかを確認する。
