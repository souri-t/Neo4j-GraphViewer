# VPN とリモートアクセスのトラブル対応

外出先から VPN につながらない場合、利用者のネットワーク、端末証明書、SSO 認証、MFA、VPN クライアント版数、アカウント状態を順に確認する。エラーメッセージが「認証に失敗しました」の場合は SSO や MFA、「接続先に到達できません」の場合は通信経路や VPN 装置を疑う。

営業部は外出先利用が多いため、端末証明書の有効期限切れ、公共 Wi-Fi の通信制限、スマートフォンテザリングの不安定さがよく原因になる。端末証明書は 1 年更新で、有効期限 14 日前から通知される。期限切れの場合は社内ネットワーク接続後に再発行する。

複数利用者で同時に接続できない場合は VPN 装置障害として扱い、ネットワーク担当へエスカレーションする。単一利用者の場合は、端末時刻、VPN クライアント再起動、証明書、MFA プッシュ通知、アカウントロックを確認する。
