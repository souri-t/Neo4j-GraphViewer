# Microsoft 365 メール配送遅延の切り分け

メールが届かない、配送が遅い、外部宛メールだけ失敗する場合、Exchange Online のサービス正常性、メールフロー、迷惑メール判定、転送ルール、メールボックス容量、送信制限を確認する。Teams は使えるがメールだけ使えない場合は、SSO 障害ではなく Exchange 側の問題である可能性が高い。

単一利用者の問題では、Outlook プロファイル破損、キャッシュモード、モバイルアプリ同期、メールボックス容量超過、ライセンス未付与を確認する。複数部署で同時に発生している場合は、Microsoft 365 広域障害、DNS、社内プロキシ、メールセキュリティゲートウェイを確認する。

利用者案内では、障害範囲、開始時刻、影響サービス、暫定回避策、次回更新予定を明記する。原因が未確定の場合は断定せず、管理センターのサービス正常性と社内ネットワークの両方を調査中と記載する。
