# 公共 Wi-Fi とプロキシによる VPN 接続不可

ホテル、空港、カフェなどの公共 Wi-Fi では、VPN 通信に必要なポートが制限されることがある。利用者からは「自宅ではつながるが出張先だけ VPN に入れない」「認証画面までは進むが接続が完了しない」と報告される。公共 Wi-Fi のログイン画面を完了していない場合も、VPN クライアントは接続先に到達できない。

確認手順は、ブラウザで通常の Web サイトを開けるか、公共 Wi-Fi の認証画面が残っていないか、スマートフォンテザリングでは接続できるか、社内プロキシ設定が固定されていないかを順に見る。VPN 以前に DNS 解決や HTTPS 通信ができない場合は、ネットワーク経路の問題として扱う。

回避策として、スマートフォンテザリング、別ネットワークへの切り替え、VPN クライアント再起動、端末時刻の同期を案内する。複数の出張者が同じ会場で接続できない場合は、会場ネットワーク側の制限として社内ポータルに注意喚起する。
