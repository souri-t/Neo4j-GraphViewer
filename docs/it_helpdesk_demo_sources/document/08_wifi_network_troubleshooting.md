# 社内 Wi-Fi とネットワーク接続トラブル

社内 Wi-Fi につながらない場合、SSID、電波強度、端末証明書、認証方式、端末時刻、IP アドレス取得、DNS 解決を確認する。ゲスト Wi-Fi と社内 Wi-Fi は認証方式が異なり、社内システムへ接続するには社内 Wi-Fi または VPN が必要である。

特定フロアだけ遅い場合はアクセスポイント障害、電波干渉、利用者集中、配線障害を疑う。単一端末だけ接続できない場合は、端末証明書、有線 LAN アダプタ、ネットワーク設定、EDR の通信制御を確認する。

社内 Wi-Fi に接続できているが共有フォルダや社内ポータルへアクセスできない場合は、DNS、プロキシ、ファイアウォール、アカウント権限を確認する。ネットワーク全体の障害が疑われる場合は、影響範囲、開始時刻、対象フロア、業務影響をまとめてネットワーク担当へ連絡する。
