# 機密部門フォルダの権限追加ルール

経理、人事、法務、経営企画の共有フォルダは機密区分が高く、通常のフォルダより厳格に権限管理する。読み取り権限、更新権限、管理者権限を分け、個人直接付与ではなく AD グループで管理する。アクセス申請には所属長とフォルダオーナーの承認が必要である。

異動者や退職者の権限削除漏れは監査指摘になりやすい。JML 手順では、異動日の翌営業日までに旧部署フォルダの権限を削除し、新部署で必要な権限だけを追加する。退職時は SSO 無効化だけでなく、共有フォルダ権限、メーリングリスト、SaaS ロールも棚卸しする。

アクセス不可問い合わせでは、承認不足、グループ反映待ち、VPN 未接続、資格情報キャッシュ、フォルダパス誤りを確認する。機密フォルダでは緊急依頼であっても承認なしの一時付与は原則禁止である。
