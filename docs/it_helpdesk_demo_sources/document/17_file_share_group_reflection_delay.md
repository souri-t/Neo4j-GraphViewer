# 共有フォルダ権限の反映待ちとキャッシュ対応

共有フォルダの権限追加後すぐにアクセスできない場合、ID 管理システムから AD グループへの反映待ち、利用者端末の資格情報キャッシュ、VPN 未接続、ファイルサーバー側の監査ログ遅延を確認する。承認済みなのに「アクセス拒否」と表示される問い合わせでは、反映待ちが原因であることが多い。

権限グループの反映には最大 30 分かかる。反映後もアクセスできない場合は、Windows からサインアウトして再ログインする、資格情報マネージャーの古い認証情報を削除する、VPN または社内 Wi-Fi へ接続し直す。経理、人事、法務のフォルダでは承認者不足により申請が保留になっている場合もある。

切り分けでは、フォルダパス、利用者 ID、所属部署、申請番号、承認状況、追加された AD グループ名、エラー時刻を確認する。権限不足とファイルサーバー障害は見た目が似るため、同じフォルダに複数利用者がアクセスできない場合は障害として扱う。
