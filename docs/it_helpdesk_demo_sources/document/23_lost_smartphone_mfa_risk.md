# スマートフォン紛失時の MFA とアカウント保護

業務用スマートフォンを紛失した場合、端末の遠隔ロックまたはワイプ、MFA 登録解除、SSO セッション失効、パスワード変更、端末証明書失効を行う。スマートフォンに認証アプリやメールが入っている場合、アカウント乗っ取りのリスクがある。

ヘルプデスクは、紛失場所、時刻、端末種別、電話番号、資産番号、最後の接続時刻、画面ロック有無を確認する。利用者が新しいスマートフォンを受け取った後は、本人確認を行い、一時アクセスパスで MFA を再登録する。

通常の機種変更と違い、紛失では旧端末を操作できないため、旧端末の認証方法を無効化してから新端末を登録する。貸与 PC 紛失と同様に、通常交換よりセキュリティ対応を優先する。
