# 日本国憲法制定過程 年表 1945-1947

この年表ノートは、国立国会図書館の主要年表に、衆議院の「日本国憲法制定経過年表（ポツダム宣言〜日本国憲法施行）」と国会図書館の詳細年表を重ね、年月日検索向けに粒度をそろえたものである。日付を起点に検索されるRAGでは、短い一覧よりも、各日の意味が分かる文脈付き年表の方が有効である。

## 1945年 戦争終結と占領開始

- 1945-07-26 米英中がポツダム宣言を発表。日本の民主化、非軍事化、基本的人権尊重への道を開く起点となる。
- 1945-07-28 鈴木貫太郎首相がポツダム宣言を「黙殺」と表明。
- 1945-08-06 広島に原子爆弾投下。
- 1945-08-08 ソ連が対日参戦。
- 1945-08-09 長崎に原爆投下。
- 1945-08-10 御前会議で国体護持を条件とするポツダム宣言受諾方針を決定。
- 1945-08-14 御前会議で受諾を最終決定。天皇が終戦の詔書を録音。マッカーサーが連合国最高司令官に就任。
- 1945-08-15 終戦の詔書放送。鈴木内閣総辞職。
- 1945-08-17 東久邇宮稔彦内閣成立。
- 1945-08-28 占領軍が本土進駐開始。GHQは横浜に設置され、のち東京へ移転。
- 1945-08-29 米政府が「降伏後ニ於ケル米国ノ初期ノ對日方針」をマッカーサーに通達。
- 1945-08-30 マッカーサーが厚木に到着。
- 1945-09-02 ミズーリ号上で降伏文書調印。無条件降伏が法的に確定。
- 1945-09-18 東久邇宮首相が、憲法改正を検討する余裕はないと表明。同日、入江俊郎が内部文書「終戦と憲法」で改正論点を整理。
- 1945-10-02 GHQ/SCAP設置。民政局などの占領統治機構が本格化。
- 1945-10-04 GHQが「自由の指令」を発令。政治犯釈放、治安維持法廃止など民主化政策が加速。マッカーサーは近衛文麿に憲法改正を示唆。
- 1945-10-09 幣原喜重郎内閣成立。
- 1945-10-11 マッカーサーが幣原首相に五大改革を要求し、「憲法の自由主義化」を示唆。
- 1945-10-25 政府に憲法問題調査委員会、いわゆる松本委員会を設置。
- 1945-11-22 近衛文麿が「帝国憲法改正要綱」を天皇に奉答。
- 1945-11-24 内大臣府廃止。近衛・佐々木の調査は制度的基盤を失う。
- 1945-12-08 松本烝治が帝国議会で憲法改正四原則を表明。
- 1945-12-15 GHQが神道指令を発出。国家神道体制の解体が進む。
- 1945-12-16 米英ソ三国外相会議がモスクワで開かれ、極東委員会設置で合意。
- 1945-12-26 憲法研究会が「憲法草案要綱」を公表。

## 1946年 1月から2月13日まで 日本側改正案からGHQ草案提示へ

- 1946-01-01 新日本建設に関する詔書、いわゆる人間宣言。
- 1946-01-04 GHQが公職追放指令。
- 1946-01-07 SWNCC228「日本の統治体制の改革」採択。天皇制改革、人権保障、文民優位などの米政府方針が具体化。
- 1946-01-21 日本自由党が憲法改正要綱を発表。
- 1946-01-30 閣議で松本私案の説明。GHQにも松本案の非公式要旨が示される。
- 1946-02-01 毎日新聞が松本委員会試案をスクープし、保守的すぎるとの批判が噴出。
- 1946-02-02 憲法問題調査委員会が甲案・乙案を決定。
- 1946-02-03 マッカーサーが三原則を提示し、GHQ民政局に憲法草案起草を命令。
- 1946-02-04 から 1946-02-10 にかけてGHQ内部で草案作成。
- 1946-02-08 日本政府がGHQへ「憲法改正要綱」を提出。
- 1946-02-13 ホイットニー民政局長が松本案を拒否し、GHQ草案を手交。ここが制定過程の最大の転換点。

## 1946年 2月14日から4月17日まで 日本案の再作成

- 1946-02-18 松本烝治が「憲法改正案説明補充」を提出し抵抗するが、GHQは再考の余地なしと回答。
- 1946-02-21 幣原首相がマッカーサーと会談。
- 1946-02-22 閣議でGHQ草案に沿う方向へ転換。
- 1946-02-25 GHQ草案日本語訳が閣僚に配付される。
- 1946-02-26 入江俊郎・佐藤達夫を中心に政府案起草開始。極東委員会は第1回会合。
- 1946-03-02 政府内で「3月2日案」完成。
- 1946-03-04 日本側がGHQへ提出。同日夜から徹夜の逐条協議が始まり、翌5日に確定案へ。
- 1946-03-05 政府が憲法改正草案を奏上。
- 1946-03-06 政府が「憲法改正草案要綱」を発表。主権在民、象徴天皇制、戦争放棄が公式に打ち出される。
- 1946-03-07 マッカーサーが同要綱を支持する声明。
- 1946-03-20 極東委員会が日本国民の世論尊重を決定。
- 1946-04-10 衆議院総選挙。女性参政権を含む新選挙法の下で実施。
- 1946-04-17 政府がひらがな口語体の「憲法改正草案」を公表し、枢密院に諮詢。

## 1946年 5月から11月まで 帝国議会審議と公布

- 1946-05-16 第90回帝国議会召集。
- 1946-05-22 第1次吉田茂内閣成立。
- 1946-05-27 内閣交代に伴い、枢密院に再諮詢。
- 1946-06-08 枢密院本会議で可決。
- 1946-06-20 帝国憲法改正案を帝国議会へ提出。
- 1946-06-21 マッカーサーが審議三原則を声明。十分な審議時間、明治憲法との法的持続性、国民の自由意思が示される。
- 1946-06-25 衆議院本会議で第1読会開始。
- 1946-06-28 衆議院帝国憲法改正案委員会に付託。委員長は芦田均。
- 1946-07-01 から 1946-07-22 まで委員会で総括質疑・逐条審議。
- 1946-07-23 修正案作成のため小委員会設置。
- 1946-07-25 から 1946-08-20 まで小委員会協議。
- 1946-08-20 芦田修正などを含む共同修正案がまとまる。
- 1946-08-21 衆議院委員会が修正議決。
- 1946-08-24 衆議院本会議で賛成421、反対8で可決。
- 1946-08-26 貴族院本会議で上程。
- 1946-08-30 貴族院特別委員会に付託。
- 1946-09-02 から 1946-09-26 まで貴族院特別委員会審議。
- 1946-09-28 貴族院でも小委員会設置。
- 1946-10-03 貴族院特別委員会が修正議決。文民条項などが加わる。
- 1946-10-06 貴族院本会議で修正可決。
- 1946-10-07 衆議院が貴族院修正に同意。
- 1946-10-12 改正案を枢密院に再諮詢。
- 1946-10-17 極東委員会が施行後再検討政策を採択。
- 1946-10-29 枢密院が最終可決。
- 1946-11-03 日本国憲法公布。公布式典実施。
- 1946-12-01 憲法普及会が組織される。

## 1947年 施行と新体制への移行

- 1947-01-03 マッカーサーが吉田首相宛書簡で、施行後再検討政策を伝達。
- 1947-04-20 第1回参議院議員通常選挙。
- 1947-04-25 第23回衆議院総選挙。
- 1947-05-02 枢密院廃止。
- 1947-05-03 日本国憲法施行。同日、附属24法律も施行。
- 1947-05-20 第1回国会召集。
- 1947-05-24 片山哲内閣成立。
- 1947-06-23 第1回国会開会式。

## 年表の読み方

この時期の検索では、同じ日付に複数の出来事が重なっている点に注意が必要である。たとえば1946-02-13はGHQ草案手交の日であると同時に、日本政府の自主的改正路線が事実上終わった日でもある。1946-03-06は「要綱発表の日」であり、国民主権・象徴天皇制・戦争放棄が対外的にも明確化した日として重要である。1946-08-24、1946-10-06、1946-10-07はそれぞれ衆議院、貴族院、衆議院再同意という別段階であり、議会修正の追跡には区別が必要である。
