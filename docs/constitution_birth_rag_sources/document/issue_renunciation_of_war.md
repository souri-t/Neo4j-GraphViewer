# 論点 戦争放棄

戦争放棄は、日本国憲法の中でもっとも象徴的な原理であり、第9条に結実した。だが制定過程を見ると、これは一度に固定された観念ではなく、占領期の非軍事化政策、日本政府内部の軍規定削除論、GHQ草案、帝国議会修正、施行後の解釈対立が積み重なって形成された論点である。

出発点はポツダム宣言の非軍事化原則である。宣言は軍国主義者の除去、戦争遂行能力の破砕、軍隊の完全武装解除、軍需産業の禁止を要求した。米国政府の初期対日方針も、武装解除を最高司令官の任務として明示していた。ただし、SWNCC228は文民部門の優位を求めたものの、軍の完全否定を必ずしも当然視しておらず、後の日本側解釈論に余地を残す部分もあった。

日本側では、終戦後の軍解体がきわめて急速に進み、1945年10月にはマッカーサーが日本軍復員完了を宣言した。この現実は、明治憲法の軍規定をどうするかという問題を生んだ。法制局内部や松本委員会では、軍規定を全面削除して非武装国家とすべきだという立場と、将来の最小限国防力を想定して議会統制付きで残すべきだという立場が鋭く対立した。最終的に2月8日提出の「憲法改正要綱」は後者寄りであり、後の第9条ほど徹底した平和主義ではなかった。

転換点は、マッカーサー・ノート第2原則である。そこには、自己の安全を保持するための戦争さえ放棄する趣旨が記されていたが、これは国際法上の自衛権まで否定するように見えたため、GHQ草案ではやや整理された形で条文化された。それでも、戦争と武力行使を永久に放棄し、戦力不保持と交戦権否認を定める条文は、当時の各国憲法の中でも際立っていた。

日本政府はGHQ草案を基に3月2日案、3月6日の要綱、4月17日の草案を作成し、第9条の文言もほぼ現行へ近づく。ここで法制局は、枢密院・帝国議会向けに想定問答を作成し、第9条は侵略・自衛を問わず戦争を放棄するが、緊急避難や正当防衛的行動まで直ちに不可能というわけではないと説明した。すでにこの段階で、自衛権と第9条の関係をどう読むかが問題化していたことが分かる。

帝国議会では、条文の主体性や自発性が問われ、衆議院で第1項冒頭に「日本国民は、正義と秩序を基調とする国際平和を誠実に希求し」が追加された。また第2項には「前項の目的を達するため」が挿入された。いわゆる芦田修正である。これにより、のちに「前項の目的を達するためでない戦力は許されるのか」という論点が生まれる。GHQや極東委員会内部でも、この修正が自衛力保持を可能にするのではないかとの見方が出たため、貴族院では第66条に文民条項が追加され、軍事的再編への警戒が条文で補強された。

その後、政府は第9条1項は侵略戦争放棄を意味し、2項が戦力保持と交戦権を否定する結果として自衛戦争もできないという理解を維持した。しかし1950年の警察予備隊、1954年の自衛隊創設に至る過程で、解釈は大きく変動する。したがって制定過程資料を読む意義は、現代の第9条論争が戦後後期に突然生まれたのではなく、1945年から1946年の文書群の中にすでに伏線がある点を確認できることにある。
