# 論点 新しい二院制議会

日本国憲法の国会制度は、明治憲法下の貴族院・衆議院から連続しているように見えて、原理的には大きく転換している。この論点では、天皇の協賛機関だった帝国議会が、国権の最高機関としての国会へ変わる過程と、その中で二院制がどう再設計されたかを追う。

明治憲法下の帝国議会は、立法権者である天皇に協賛する機関であり、貴族院は皇族・華族・勅任議員など非公選議員を含む身分制的な第二院だった。両院の権限は概ね対等で、衆議院優越は限定的だった。敗戦後、国民主権へ転換する以上、この構造は維持できない。議会制度の民主化は、憲法改正論の初期段階からすでに重要課題となっていた。

近衛文麿に対してアチソンが示した改正項目には、衆議院権限拡大、貴族院拒否権の撤廃、議会責任原理の確立、貴族院民主化が含まれていた。またGHQ内部研究やSWNCC228でも、立法部は全員が公選議員によって組織されるべきだという方向が示されている。ここから、少なくとも旧来の貴族院をそのまま残す選択肢は早い段階でほぼ失われていた。

日本政府側の松本委員会は、それでも二院制自体は維持すべきだと考えていた。名称として「上院」「第二院」「参議院」などが検討され、最終的には参議院が有力になる。甲案では、参議院と衆議院の両院制を採りつつ、法律・予算で衆議院優越を認め、参議院は選挙または勅任議員で構成するという折衷案が示された。つまり、旧貴族院を民主化して残す発想が中心だった。

GHQ内部では一院制も真剣に検討された。1946年2月5日の民政局会合では、簡明性や日本政治の発達段階を理由に一院制が望ましいという意見が優勢だった。しかし、二院制か一院制かは、日本政府との交渉でより重要な原則を守るための譲歩材料にもなり得た。2月13日のGHQ草案では一院制国会とされたが、日本側は政治の安定性や継続性を理由に強く二院制を主張する。結果として、国民主権と公選制を前提とするなら二院制を認めてもよいという方向へ動いた。

3月以降の政府案では、両院とも全国民を代表する公選議員で組織されることになり、二院制は残っても、旧貴族院的な身分制第二院は否定された。帝国議会審議では、金森徳次郎が参議院の役割を、衆議院への抑制と慎重審議の機能に求め、構成面では知識経験者をどう反映させるかが議論された。職能代表制への関心も強かったが、最終的には明確な職能院構想には進まず、公選議員による参議院へ落ち着く。

この論点の意義は、単に「二院制が採用された」という事実ではなく、二院制の意味が身分制的第二院から民主的熟議院へ置き換えられた点にある。衆議院優越の範囲、参議院の独自性、全国民代表原理は、その後の国会制度論の基礎になった。
