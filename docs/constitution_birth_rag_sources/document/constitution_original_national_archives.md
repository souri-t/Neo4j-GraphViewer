# 日本国憲法原本 国立公文書館資料ノート

このファイルは、国立公文書館デジタルアーカイブに掲載された「日本国憲法・御署名原本・昭和二十一年・憲法一一月三日」のメタデータ、展示解説、画像導線をまとめたものである。RAG用途では、公布原本の所在、請求番号、年月日、原本閲覧可否、関連展示をひとまとまりで検索できるようにすることが目的である。

対象資料は、国立公文書館デジタルアーカイブの簿冊「日本国憲法・御署名原本・昭和二十一年・憲法一一月三日」で、請求番号は `御30168100` である。記述レベルは file、移管元は `＊内閣・総理府`、作成・取得者は内閣、年月日は1946-11-03、媒体は紙、保存場所は本館とされる。利用制限区分は `公開 / 原本閲覧（否）` であり、メタデータの利用条件は `CC0` である。

この資料は、日本国憲法の公布原本であり、天皇の裁可と御署名、関係国務大臣の副署を伴う正式原本として位置づけられる。デジタルアーカイブの資料内容欄には、公布文に続いて、裕仁、吉田茂、幣原喜重郎、木村篤太郎、田中耕太郎、和田博雄、石橋湛山、金森徳次郎などの関係者名が掲載されている。関連事項欄には「日本国憲法の公布原本。昭和22年5月3日施行。」とあり、公布日と施行日を原本資料の側から確認できる。

画像情報としては、デジタルアーカイブの公開ページにサムネイル導線が存在し、現時点で確認できるサムネイルパスは次のとおりである。

- サムネイルURL: `https://www.digital.archives.go.jp/thumb/142415/C100189145000.jpg`
- デジタルアーカイブ資料ページ: `https://www.digital.archives.go.jp/file/142415`

また、国立公文書館本体サイトのデジタル展示「誕生 日本国憲法」では、この原本そのものだけでなく、公布原議書、枢密院会議記録、金森徳次郎関係資料、憲法普及会資料など、原本に至る前後の行政文書群が一続きで示されている。したがって、原本単体を確認するだけでなく、展示の第II部・第III部と接続して読むと、公布・普及・施行の全体像が把握しやすい。

実物展示については、国立公文書館が憲法記念日前後などに原本特別展示を行うことがあり、通常展示では複製展示である場合がある。常設・特別展の情報は年度によって変動するため、現地展示状況は別途最新情報で確認するのが望ましいが、少なくともデジタルアーカイブ上のメタデータと資料ページURLは、原本参照の安定した基準点になる。

RAGで重要なのは、`御30168100`、`1946-11-03`、`1947-05-03`、`公開 / 原本閲覧（否）`、`内閣`、`国立公文書館デジタルアーカイブ`、`公布原本`、`御署名原本` を検索語として紐付けておくことである。公布・施行・原本の三者を混同しないように整理しておくと、年月日照会や原本確認系の質問に強くなる。
