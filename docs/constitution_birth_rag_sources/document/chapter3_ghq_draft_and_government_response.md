# 第3章 GHQ草案と日本政府の対応

この章は、日本国憲法制定過程の中核である。1946年2月13日にGHQが日本政府案を拒否して自らの草案を手交した出来事は、国立国会図書館が言うとおり全過程のターニング・ポイントであり、RAGでも最重要日付の一つになる。

背景には、対日占領管理方式の変化があった。1945年末のモスクワ三国外相会議で極東委員会設置が合意され、1946年2月26日からは憲法問題に対するGHQの裁量が狭まる見通しになっていた。これに加え、1月7日にSWNCC228「日本の統治体制の改革」が採択され、天皇制改革、人権保障、議会民主化、文民優位といった米国政府の方針がマッカーサーに示された。GHQとしては、極東委員会による外部統制が本格化する前に、占領改革の基本方向を憲法へ埋め込む必要があった。

2月1日の毎日新聞による松本委員会試案スクープは、こうした事情のもとで強い触媒になった。日本側案が保守的すぎると判断したホイットニー民政局長は、GHQ自身による草案起草を進言し、2月3日、マッカーサーは三原則を提示した。通例「マッカーサー・ノート」と呼ばれるこの原則は、天皇の地位、戦争放棄、封建制度の撤廃を骨格とする。翌4日から民政局内のチームが短期間で起草作業を行い、2月13日には完成草案が日本側に手渡された。

2月13日の会見で、ホイットニーは松本要綱を拒否し、松本烝治・吉田茂らにGHQ草案を示した。これは日本政府に対する「共同起草の提案」というより、実質的にはこの方向で改正案を作れという強い要求だった。後に松本は補足説明を提出して抵抗するが、GHQは受け入れなかった。ここで政府は、明治憲法の小幅改正路線を断念し、GHQ草案に沿った再起草へ進む。

転換後の日本側起草は、入江俊郎と佐藤達夫が中心になって進めた。2月27日から極秘で作業が始まり、3月2日に試案がまとまる。3月4日、松本と佐藤がGHQに提出するが、その日のうちに民政局との激しい逐条協議に入り、徹夜交渉の末、3月5日に確定案が成立した。この段階で、日本政府はGHQ草案を単純翻訳したのではなく、法制技術・文言・制度配置を日本の法体系に馴染ませる作業を行っている。

その成果が3月6日の「憲法改正草案要綱」であり、さらに口語体への条文化を経て4月17日の「憲法改正草案」になる。主権在民、象徴天皇制、戦争放棄、基本的人権保障、二院制国会、地方自治など、現行憲法の主要要素はこの時点でほぼ形を整えた。もっとも、前文・第1条・第9条・第24条などは、その後の議会修正によってさらに重要な変化を受ける。

もう一つ見落とせないのが、マッカーサーと極東委員会の対立である。3月6日の要綱発表は米国政府にとっても突然で、GHQが日本政府案に深く関与したことは極東委員会を刺激した。極東委員会は選挙延期や係官派遣を求めたが、マッカーサーは介入を拒み、日本側の審議過程を先に進めた。したがってこの章は、日米関係だけでなく、米国政府内部と対日占領機構内部の権力関係を読む章でもある。
