# 第4章 帝国議会における審議

この章は、1946年6月から10月までの帝国議会審議を扱う。日本国憲法は、明治憲法73条の改正手続を用いて制定されたため、枢密院、衆議院、貴族院という旧体制の制度を経由しながら、新しい統治原理を組み込むという独特の経路をたどった。RAGでは「誰がどこでどの文言を変えたか」を追えるようにしておくと有効である。

まず、4月17日に公表された「憲法改正草案」は枢密院に諮詢されたが、4月22日の幣原内閣総辞職により一旦撤回され、吉田内閣成立後の5月27日に修正のうえ再諮詢された。6月8日、枢密院本会議はこれを可決する。ここでの審議は公開政治よりは法制・統治原理の整合性確認に近いが、のちの議会審議に備えた重要な中間関門だった。

衆議院では、4月10日の総選挙で女性参政権が導入された直後という政治状況が大きい。6月20日、帝国憲法改正案が勅書をもって提出され、6月28日には芦田均を委員長とする帝国憲法改正案委員会へ付託された。7月1日から逐条審議が始まり、7月23日には修正案作成のため小委員会が設置される。ここで各党の修正要求とGHQからの非公式意向が交錯し、現行憲法に大きな影響を与える文言修正が練られた。

もっとも有名なのが第9条の芦田修正である。8月20日、小委員会は第9条2項冒頭に「前項の目的を達するため」を挿入する修正を含む共同案を取りまとめた。この文言は後年、自衛力保持の可否をめぐる解釈論の出発点となる。また、前文や第1条では国民主権表現が強められ、第24条では家族関係における個人の尊厳と両性の本質的平等が整えられた。8月24日、衆議院本会議は賛成421、反対8で改正案を可決する。

貴族院審議も単なる追認ではなかった。8月30日に安倍能成を委員長とする特別委員会へ付託され、9月2日から審議、9月28日には小委員会が設置される。ここでGHQ側の要請に沿って「文民条項」が重要修正として浮上し、最終的に第66条2項に「内閣総理大臣その他の国務大臣は、文民でなければならない」が加えられる。これは第9条解釈をめぐるGHQや極東委員会の警戒の反映でもある。

10月6日、貴族院本会議は修正案を可決し、翌7日、衆議院がこれに同意した。その後、10月12日の枢密院再諮詢、10月29日の最終可決を経て、11月3日に日本国憲法が公布される。したがって、公布条文は4月17日草案でも2月13日GHQ草案でもなく、帝国議会と枢密院を通過した最終形であることを押さえる必要がある。

この章では、国内政治だけでなく極東委員会の影響も追うべきである。7月2日、極東委員会は「日本の新憲法についての基本原則」を決定し、主権在民、普通選挙、文民統制、貴族院廃止方向などを改めて打ち出した。GHQはこの意向も踏まえつつ日本政府へ修正を働きかけたため、議会修正の背後には占領当局内の力学も存在した。
