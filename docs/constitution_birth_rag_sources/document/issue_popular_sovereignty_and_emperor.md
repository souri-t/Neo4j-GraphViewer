# 論点 国民主権と天皇制

日本国憲法制定過程で最大の争点の一つが、主権の所在と天皇の地位をどう定めるかだった。この論点は、単に第1条の文言をめぐる技術的問題ではなく、敗戦後もなお「国体護持」を求める日本側と、民主化の原理を憲法に埋め込みたいGHQ側との正面衝突の場であった。

ポツダム宣言受諾時、日本政府は天皇の国家統治大権に変更を加えないことを前提に受諾したいと考えた。いわゆる「国体護持」である。これに対し連合国側は、天皇の権限は最高司令官の制限下に置かれ、日本の最終的な政治形態は日本国民の自由意思によって決まると回答した。この時点で、天皇制存続は自明ではなく、少なくとも民主主義的改革を受ける対象として扱われた。

米国政府のSWNCC228は、天皇制を廃止するか、存続させるとしても内閣の助言に基づく民主的君主制へ改めるか、いずれかの方向を示していた。他方、マッカーサー自身は、占領統治を円滑に進めるため天皇制存続をほぼ決めていたとされる。ここで重要なのは、米国側の課題が「存続か廃止か」だけではなく、「存続させるなら主権との関係をどう書き換えるか」だった点である。

日本政府側の松本委員会は、当初、天皇統治権の総攬という明治憲法の基本原則を変更する意思を持たなかった。松本四原則や甲案・乙案も、この前提を崩していない。これに対し、民間草案や政党案では差が大きい。共産党案は人民主権と天皇制廃止、憲法研究会案は主権在民と天皇権能の儀礼化、社会党案は国民共同体的な主権論と天皇制維持をそれぞれ示した。つまり日本側内部でも、この論点は単一ではなかった。

GHQ草案作成で決定的だったのが、2月3日のマッカーサー・ノートである。そこでは天皇の地位を「元首」としつつ、権能は憲法に基づき人民意思に応えるという原則が示された。さらに民政局内部の検討で、当初は主権在民を別条文で明記する案や、「天皇が君臨する」といった表現も試みられたが、日本語で統治権を連想させるとして整理され、最終的に「天皇は日本国の象徴であり日本国民統合の象徴」とする条文へ収斂した。ここに象徴天皇制の核心がある。

ただし日本政府が最初に作成した3月2日案では、前文が外され、第1条も「国民至高ノ総意」といった曖昧な表現になり、主権の所在がぼかされた。GHQとの再交渉で前文は復活し、第1条もGHQ案に近づいたが、なお「総意」表現は残った。帝国議会ではこの点が激しい争点となり、金森徳次郎は「国体は変わらず、変わったのは政体である」と答弁した。この説明はGHQ側には不十分と受け取られ、さらに国民主権を明確にする修正が求められる。

その結果、衆議院では前文に「主権が国民に存することを宣言し」が入り、第1条も「主権の存する日本国民の総意に基く」と修正された。これは現行憲法の国民主権規定の最終形であり、制定過程の妥協点でもある。すなわち、天皇制は存続したが、その正統性の根拠は天皇固有の統治権ではなく、主権者たる国民の総意へ移された。
