# 第1章 戦争終結と憲法改正の始動

この章は、敗戦から松本委員会設置までの局面を扱う。日本国憲法の制定史では、1945年8月の降伏から1945年10月末までのわずかな期間に、占領政策の基本方向、天皇制をめぐる米国側の判断、政府内部の初期検討がほぼ同時並行で進む。後のGHQ草案提示より前に、どのような条件設定がなされていたかを理解するための基礎資料である。

米国側では、日米開戦直後から戦後日本処理の検討が始まっていた。国務省内には、天皇制を存置しつつ既存の統治機構を通じて改革を進める案があり、これに対して軍部には、天皇制廃止や徹底的な制度改革を主張する議論もあった。最終的にポツダム宣言は、天皇制存続を明言しないまま、日本の民主化と非軍事化を要求する曖昧だが強力な枠組みを示した。この曖昧さが、敗戦後の日本側に「国体護持」を守りながら改革に応じる余地があるのではないかという期待を持たせた。

日本政府はポツダム宣言受諾に際し、天皇の国家統治大権に変更を加えないことを条件として理解したいと申し入れた。しかし連合国側の回答は、最終的な政治形態は日本国民の自由意思に委ねられるというものであり、ここに天皇主権体制の将来が外部条件に従属する構図が生まれた。1945年8月から9月にかけては、政府も世論もまだ憲法改正より占領対応に追われていたが、すでに法制局や外務省では、明治憲法のどこが占領政策と衝突するかを洗い出す作業が始まっていた。

とりわけ重要なのが、入江俊郎らによる内部文書「終戦と憲法」である。ここでは、軍規定、天皇大権、人権保障、議会制度など、後に改正論点となる項目がほぼ出そろっている。つまり、政府内にはかなり早い段階で「どこが問題になるか」の認識が存在した。ただし、東久邇宮内閣は政治的にも時間的にも憲法改正に踏み込めず、実際の政策化には至らなかった。

局面を動かしたのは、1945年10月のGHQによる民主化指令である。10月4日の「自由の指令」は、治安維持法などを廃止させ、政治犯の釈放と思想統制の撤廃を命じた。これは単なる人権政策ではなく、明治憲法体制を支えてきた統治技術を崩す措置だった。同じ日にマッカーサーは近衛文麿に憲法改正を示唆し、10月11日には幣原喜重郎首相に対しても「憲法の自由主義化」に言及した。占領改革と憲法問題がここで明確に接続される。

近衛文麿は、この示唆を受けて内大臣府で独自調査を始め、佐々木惣一も協力した。しかし、内大臣府が憲法問題を扱うことには憲法上の疑義があり、また近衛自身の戦争責任もあって、国内外の批判が強かった。その反動として、政府自身が正面から憲法問題に対応する必要が高まり、10月25日に松本烝治を委員長とする憲法問題調査委員会、いわゆる松本委員会が設置される。ここで憲法改正問題は、非公式研究から正式な政府調査へと移行した。

RAG的には、この章は「起源」の章であり、後続のすべての論点の前提条件を与える。検索で重要なのは、ポツダム宣言、バーンズ回答、自由の指令、終戦と憲法、東久邇宮内閣、幣原内閣、近衛文麿、佐々木惣一、入江俊郎、松本委員会という語の結びつきである。天皇制存続の可否、軍規定削除、人権保障改革、議会民主化などは、すでにこの段階で論点化していた。
