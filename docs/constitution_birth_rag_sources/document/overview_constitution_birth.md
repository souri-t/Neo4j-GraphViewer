# 日本国憲法の誕生 全体概要

このノートは、国立国会図書館「日本国憲法の誕生」の全体構成を土台に、国立公文書館のデジタル展示と衆議院の参考資料を重ねて、日本国憲法成立過程の大きな流れを一つの長文にまとめたものである。RAG用途では、個別の日付や人物名だけでなく、「どの局面で何が転換点になったのか」を掴める全体文書が必要になるため、細部よりもまず過程全体の骨格を明示する。

日本国憲法の成立は、敗戦後に外から加えられた占領改革の圧力と、戦前の統治体制では民主化を実現できないという国内側の危機意識とが重なって進んだ。国立国会図書館の「はじめに」が強調するのは、憲法制定を単純な押し付けか自主制定かの二分法で捉えるのではなく、占領政策、政府内の調査、民間の改正案、帝国議会での修正、公布後の普及活動までを含む複合的な政治過程として見る必要があるという点である。

全体像は、1946年2月13日を境に大きく二段階に分けると理解しやすい。第一段階は、1945年10月のマッカーサーによる「憲法の自由主義化」の示唆から、日本政府が独自に明治憲法改正を検討し、1946年2月8日に松本委員会の「憲法改正要綱」をGHQへ提出するまでである。この段階では、日本政府は明治憲法の法的連続性を前提にしつつ、天皇大権を一部整理し、議会や内閣の機能をやや強める程度の改正を構想していた。近衛文麿・佐々木惣一の調査、松本四原則、甲案・乙案、政党案、憲法研究会案などがここに含まれる。

第二段階は、1946年2月13日にGHQが日本側案を拒否し、自ら作成したGHQ草案を提示してから、帝国議会の審議、公布、施行に至るまでである。この局面で主権の所在、天皇の地位、戦争放棄、基本的人権、議会制度といった基本原理が一気に組み替えられた。もっとも、GHQ草案がそのまま現行憲法になったわけではない。3月2日案、3月5日案、3月6日の「憲法改正草案要綱」、4月17日の口語体「憲法改正草案」、衆議院の芦田修正、貴族院での文民条項追加など、各段階で日本側の翻案と政治的調整が積み重なっている。

重要なのは、この制定過程が単なる文書作成作業ではなく、戦後日本国家の再建構想そのものだったことである。ポツダム宣言が求めた民主化と非軍事化は、天皇主権の維持を前提とする明治憲法体制と緊張関係にあった。そこで、日本政府は「国体護持」を守りつつ占領政策に対応しようとし、GHQは象徴天皇制・国民主権・平和主義・人権保障を軸に新しい統治原理を確立しようとした。帝国議会では、こうした原理を国内政治の言葉に置き換える作業が進み、前文や第1条の国民主権表現、第9条の文言、第24条の家族観、第66条の文民条項などに結晶した。

公布後も過程は終わらない。1946年11月3日の公布、1947年5月3日の施行のあいだには、附属法制の整備、新しい皇室典範や国会法・内閣法・裁判所法・地方自治法の制定、憲法普及会による啓発、学校教育や小冊子配布などを通じて、新憲法を社会に根付かせるための国家的な普及事業が行われた。国立公文書館の展示は、この公布後の普及局面まで含めて「誕生」を描いている点が有用である。

RAGの観点では、全体像を支える検索軸として、少なくとも次の語群を押さえておくとよい。ポツダム宣言、SWNCC228、自由の指令、近衛文麿、佐々木惣一、松本烝治、松本四原則、憲法研究会、マッカーサー三原則、GHQ草案、3月2日案、憲法改正草案要綱、4月17日草案、枢密院、芦田修正、文民条項、公布、施行、憲法普及会。これらは章別資料や論点別資料へのハブになる。
