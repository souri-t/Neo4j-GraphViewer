# 論点 基本的人権の保障

基本的人権保障の拡充は、日本国憲法の民主化をもっとも端的に示す部分である。制定過程では、明治憲法の「法律ノ範囲内」による権利保障をどう乗り越えるか、社会権をどこまで導入するか、外国人にも権利保障を及ぼすかが主要論点となった。

明治憲法下の権利規定は、臣民の自由や権利を認めつつも、多くが「法律ノ範囲内」で保障されていた。これは立法によって権利保障を広く制約できる構造を意味し、占領改革が求めた民主主義や人権尊重と衝突した。米国政府のSWNCC228やGHQ内部研究でも、この点は明確な欠陥として認識されていた。日本側でも、近衛案や松本四原則などは権利保障拡張の必要性を認めていたが、制約原理そのものを根本から変えるには至らなかった。

GHQ草案の特色は、個別権利条文に一律の「法律ノ範囲内」を付けず、代わって「一般の福祉」ないし公共の利益との調整原理を採った点にある。日本政府の3月2日案以降は、これが「公共ノ福祉」という言葉で受け継がれ、現行憲法の権利体系の基礎になった。施行後、この「公共の福祉」が旧憲法的な制限原理とどう違うのかは争点となり続けるが、制定時点では「法律が権利を自由に圧縮する仕組み」からの転換として理解できる。

社会権の導入も大きい。明治憲法に存在しなかった勤労権、労働基本権、教育を受ける権利、生存権的発想が、GHQ草案と日本側の交渉を経て具体化した。ワイマール憲法や日本の社会立法思想の影響を背景に、民間草案や社会党案、憲法研究会案でも社会権は重視されていた。4月17日草案では条文化がやや簡略化されたが、衆議院審議で第25条の生存権規定が加えられたことで、現行憲法の福祉国家的性格が強まった。

外国人の人権も注目点である。米国側は、明治憲法が外国人の権利保障を想定していないことを問題視した。GHQ草案では「何人」という主語や、外国人への平等保護に近い発想が盛り込まれたが、日本政府はしばしば「国民」へ引き戻そうとした。交渉の過程で、「何人」と「国民」の使い分けが条文ごとに調整され、現行憲法のテキストが固まっていく。施行後、外国人にも性質上可能な人権規定は及ぶという解釈が広がるが、その出発点はこの制定過程にある。

また、第24条の家族条項も人権保障史の観点から重要である。婚姻の成立を両性の合意のみに基づかせ、家族法を個人の尊厳と両性の本質的平等に立脚して制定すべきだとしたことは、家制度中心の旧民法秩序に対する決定的な転換だった。これは単に家族法改革の起点であるだけでなく、人権保障を私的領域まで広げる条文として読む必要がある。

RAGでは、法律ノ範囲内、公共の福祉、生存権、労働基本権、教育権、外国人の人権、何人、国民、第24条、個人の尊厳、両性の本質的平等を主要検索語として持つと有用である。
