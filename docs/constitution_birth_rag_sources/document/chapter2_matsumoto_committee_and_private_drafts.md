# 第2章 松本委員会と民間草案

この章は、1945年末から1946年初頭にかけて、日本政府と民間社会がそれぞれどのような改正構想を出したかを扱う。RAGでこの章が重要なのは、「GHQ草案以前に日本側は何を考えていたのか」という問いに答えるためであり、近衛・佐々木案、松本委員会の甲案・乙案、憲法研究会案、政党案を並べて比較できることに価値がある。

近衛文麿と佐々木惣一による内大臣府調査は、GHQの示唆を受けて始まったが、あくまで天皇制維持を前提に、統治機構の再編を模索する色合いが強かった。1945年11月22日、近衛は「帝国憲法ノ改正ニ関シ考査シテ得タル結果ノ要綱」を奉答し、翌24日には佐々木惣一も「帝国憲法改正ノ必要」を奉答した。いずれも、明治憲法秩序を根底から転換するというより、戦後の現実に合わせて調整する発想が濃い。

他方、政府の松本委員会は、当初こそ「調査研究」を主眼に置いていたが、占領改革の進行に伴い、現実に改正案を準備せざるを得なくなる。松本烝治は1945年12月8日、帝国議会答弁の形で、いわゆる松本四原則を示した。要点は、天皇統治権の基本原則は維持しつつ、議会権限の拡大、国務大臣の輔弼責任の明確化、国民の権利保障の拡張を検討するというものである。ここから分かるのは、日本政府がなお「天皇主権を前提にした改良」を構想していたことだ。

1946年に入ると、松本委員長自身の私案をもとに、宮沢俊義らの協力を受けながら「憲法改正要綱（甲案）」が整理される。さらに大幅改正も想定して「乙案」も用意された。ただし、いずれも天皇の地位を根底から変えるものではなく、国民主権、象徴天皇制、戦争放棄といった後の現行憲法の核心には届いていない。2月1日に毎日新聞がこの政府試案を報じると、保守的で現状維持的との批判が集中した。

一方で、民間側の草案群ははるかに多様だった。最も有名なのが、1945年12月26日に発表された憲法研究会の「憲法草案要綱」である。この案は、天皇の権能を国家的儀礼に限定し、主権在民、生存権、男女平等などを明示しており、現行憲法の基本原理に近い内容をすでに備えていた。GHQ内部でも注目され、のちの草案作業に少なからぬ示唆を与えたとされる。

政党案も比較材料として重要である。自由党・進歩党は、明治憲法の基本を維持しつつ一部修正にとどまる傾向が強かった。社会党案は、生存権や社会権を意識しつつ天皇制を一定程度維持する構想を示した。共産党案は、人民主権と天皇制廃止を強く打ち出した。つまり、1945年末から1946年初頭の日本社会には、保守的改良、象徴化、社会国家化、共和制志向まで、幅広い制度イメージが共存していた。

この章の要点は、GHQ草案が突然無から生まれたわけではなく、日本側の政府案・民間案・政党案との比較の中で、その急進性や革新性が見えてくる点にある。松本委員会は天皇主権を維持しようとし、憲法研究会は主権在民と象徴天皇制に近づき、政党案は党派ごとに民主化の深さが異なった。こうした差異を押さえておくと、2月13日のGHQ草案提示がなぜ「転回」と呼ばれるのかが理解しやすい。
