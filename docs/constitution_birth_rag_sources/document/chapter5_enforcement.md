# 第5章 憲法の施行

この章は、1946年11月3日の公布から1947年5月3日の施行、さらに第1回国会と憲法普及活動までを扱う。制定史の資料はしばしば草案作成や議会審議で終わりがちだが、RAG用途では「公布後に何が行われたか」を別文書で持つことに意味がある。新憲法は、条文が公布されただけでは機能せず、その実施法・教育・広報・新選挙を通じてはじめて制度として動き出したからである。

公布は1946年11月3日、施行は1947年5月3日である。この半年のあいだに、新憲法に適合する法体系へ移るための大規模な附属法制整備が進められた。新しい皇室典範が法律として制定され、国会法、内閣法、裁判所法、地方自治法などの統治基本法が新設された。民法や刑法の改正も、家族制度や人権保障の観点から重要である。つまり、施行とは単に発効日を迎えることではなく、統治全体を新しい憲法原理へ接続する法技術の集積だった。

政治制度の切替も並行して進む。1947年4月20日に第1回参議院議員通常選挙、4月25日に第23回衆議院総選挙が実施され、新憲法の下で最初の国会を構成する議員が選ばれた。5月20日には第1回国会が召集され、5月24日には片山哲を首班とする内閣が成立、6月23日には開会式が行われた。ここで、旧帝国議会は姿を消し、国会・内閣・裁判所から成る新憲法体制が実際に始動する。

普及活動も大規模だった。1946年12月1日には、憲法の精神を国民へ浸透させるため「憲法普及会」が組織される。会長は芦田均で、中央組織の下に都道府県支部が置かれた。活動内容は講演会、解説書刊行、懸賞論文、映画、歌唱教材など多岐にわたる。とくに『新しい憲法 明るい生活』や『あたらしい憲法のはなし』のような冊子は、学校教育や家庭向け普及に大きな役割を果たした。国立公文書館の展示は、この公布後の普及実務を資料群として追える点が有用である。

施行後すぐに再検討問題が持ち上がった点も見逃せない。極東委員会は1946年10月17日、「施行後1年を経て2年以内に新憲法を再検討する」との政策を決定していた。マッカーサーは1947年1月、この方針を吉田首相へ伝える。しかし日本国内では、憲法再検討よりも新制度の運用開始が優先され、再検討は現実の政治日程にならなかった。1949年5月、極東委員会は改正要求を断念する。

この章の検索語としては、公布、施行、憲法普及会、新しい憲法 明るい生活、あたらしい憲法のはなし、第1回参議院選挙、第23回衆議院総選挙、片山哲内閣、附属24法律、再検討政策が重要である。制定史の終点であると同時に、戦後日本の新しい政治社会が始まる起点でもある。
