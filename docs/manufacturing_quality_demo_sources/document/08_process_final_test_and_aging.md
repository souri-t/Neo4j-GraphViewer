# 最終検査とエージング試験

最終検査では、起動確認、電池電圧、無線送信、RSSI、温湿度測定、ファームウェア版数、校正係数、外観を確認する。検査治具はシリアル番号を読み取り、各測定値を品質記録へ自動登録する。

エージング試験では、25 度環境で 12 時間の連続動作を行い、再起動ログ、欠測、電圧低下、測定値の急変を監視する。ロット A23 の電源不具合後は、無線送信時の電圧降下を検出する短時間負荷試験を追加した。

温度測定値のずれを防ぐため、最終検査では校正係数が空欄でないこと、ファームウェア版数が出荷許可リストに含まれることを確認する。治具通信エラーが発生した場合は再検査を必須とし、暫定値のまま合格登録できないようにした。
