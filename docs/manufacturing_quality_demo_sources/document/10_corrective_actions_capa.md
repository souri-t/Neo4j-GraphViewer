# 是正処置と再発防止 CAPA

重大不具合または同一原因の反復不具合が確認された場合、品質保証部は CAPA を起票する。CAPA では、不具合現象、対象範囲、暫定処置、根本原因、恒久対策、効果確認、責任者、期限を明確にする。

ロット A23 の電源不良では、対象在庫の隔離、出荷済みシリアル番号の抽出、代替部品への切替、受入検査の強化、最終検査への負荷試験追加を実施した。温度測定値ずれでは、校正ログ未登録品の出荷停止、治具通信エラー時の再検査必須化、ファームウェア補正テーブル修正を行った。

再発防止策は実施して終わりではなく、3 ロット連続で同種不具合が発生しないことを効果確認の基準とする。効果確認には、工程内不良率、市場返却率、顧客クレーム件数、検査ログの異常件数を用いる。
