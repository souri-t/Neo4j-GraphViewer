# 顧客クレームと市場返却品の一次解析

顧客クレームを受けた場合、営業または保守担当は現象、発生日時、設置場所、使用期間、シリアル番号、写真、ログを収集する。品質保証部は受付番号を発行し、重大度を判定する。安全性、出荷停止の可能性、複数顧客への波及がある場合は品質会議へ即日エスカレーションする。

市場返却品は、開封前の外観、梱包状態、電池状態、設置痕を記録してから解析する。起動しない品は電源ライン測定、通信不良品は RSSI と再起動ログ、温度ずれ品は恒温槽での三点測定を行う。返却品で再現しない場合は、顧客環境のゲートウェイ配置、金属棚、温度分布、電波ノイズを確認する。

解析結果は不具合報告書にまとめ、関連する工程記録、部品ロット、ファームウェア版数、過去の類似案件と紐づける。顧客回答には、確認済み事実、推定原因、暫定対応、恒久対応予定を分けて記載する。
