# サプライヤー部品ロット A23 のコンデンサ不具合

部品ロット A23 の 10uF セラミックコンデンサについて、納入後の抜取検査では規格内だったが、実装後の熱衝撃と基板曲げで容量低下が発生する事例が確認された。対象部品は QS-100 の電源基板で昇圧 IC 周辺に使用され、起動しない、再起動を繰り返す、無線送信時だけ電圧が落ちるといった現象につながった。

調査では、受入検査の測定結果、サプライヤーの製造日、梱包状態、SMT リフロー条件、実装後検査の不良履歴を照合した。A23 は通常ロットより端面電極のばらつきが大きく、リフロー後の機械的ストレスに弱い傾向があった。

暫定処置として A23 使用基板を隔離し、出荷済み品は対象シリアル番号リストを営業と保守へ共有した。恒久処置として代替部品へ切り替え、受入検査に基板曲げ後の容量測定を追加し、サプライヤーには 8D レポートを要求した。
