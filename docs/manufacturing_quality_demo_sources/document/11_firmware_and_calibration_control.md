# ファームウェアと校正係数の管理

QS-100 のファームウェアは、無線送信、電源監視、温湿度補正、再送制御を担当する。出荷可能な版数は品質保証部が承認し、最終検査治具の出荷許可リストに登録する。未承認版や評価版が書き込まれた個体は出荷検査で不合格になる。

ファームウェア 1.1 系では再送キュー満杯時に送信復帰が遅れる問題があり、無線通信切断の調査で確認対象になる。1.2.0 では温度補正テーブルの丸め処理に問題があり、高温側で測定値が上振れする事例があった。修正版では再送キューの解放処理と補正テーブル計算を見直した。

校正係数は個体校正工程で生成され、シリアル番号とともに品質記録へ保存される。治具通信エラーで係数登録が失敗した場合、暫定係数のまま出荷されないよう最終検査で空欄チェックを行う。
