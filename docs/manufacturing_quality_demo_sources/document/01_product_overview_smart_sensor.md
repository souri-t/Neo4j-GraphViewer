# 小型環境センサー QS-100 製品概要

QS-100 は工場や倉庫の温度、湿度、振動を測定し、920MHz 無線でゲートウェイへ送信する小型環境センサーである。主な構成部品は温湿度センサーモジュール、無線モジュール、電源基板、リチウム電池ホルダー、樹脂ケースである。

製品は受入検査、SMT 実装、手はんだ補修、ファームウェア書き込み、個体校正、エージング、出荷検査を経て出荷される。品質記録では製造ロット、部品ロット、作業ライン、検査治具、ファームウェア版数、校正係数を紐づけて管理する。

過去に多い問い合わせは、温度測定値のずれ、無線通信の断続的な切断、電源投入後に起動しない不具合である。現象だけでは原因を特定できないため、発生環境、ロット、ファームウェア版数、最終検査結果、顧客先の設置条件を合わせて確認する。
