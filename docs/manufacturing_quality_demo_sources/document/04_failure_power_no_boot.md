# 電源投入後に起動しない不具合

電池を入れても LED が点灯せず起動しない場合、電池極性、電池ホルダー接点、電源基板の昇圧 IC、はんだクラック、保護回路の作動を確認する。出荷直後の不具合では輸送振動による接点浮き、長期保管品では電池電圧低下が多い。

ロット A23 では特定サプライヤーのセラミックコンデンサに微細クラックが入り、温度変化後に昇圧回路が不安定になる事象が確認された。外観検査では見逃しやすく、X 線検査または基板曲げ後の通電試験で再現することがある。

一次切り分けでは良品電池で再確認し、電源投入時の突入電流、昇圧 IC 出力、基板上の 3.3V ラインを測定する。電源基板起因が疑われる場合は同一製造ロットの在庫を隔離し、部品ロットと実装ラインを追跡する。
