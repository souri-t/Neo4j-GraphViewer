# はんだ実装と組立工程の管理

QS-100 の実装工程では SMT リフロー後に外観検査を行い、アンテナコネクタ、電池ホルダー、センサー周辺は手はんだ補修の対象になる。手はんだ部位では、濡れ不足、ブリッジ、過加熱、フラックス残渣、はんだクラックが主な不良モードである。

無線通信切断が増えたロットでは、アンテナコネクタの半挿しとシールドケースへの接触が確認された。電源起動不良では、電池ホルダー端子のはんだ量不足により輸送振動後に接点抵抗が上がる事例があった。温度測定値のずれは、センサーモジュール周辺のフラックス残渣やケース内部の熱こもりと関連することがある。

工程内対策として、作業標準書に写真付きの良否判定例を追加し、アンテナコネクタ挿入後の引張確認、電池ホルダーの導通検査、センサー周辺の洗浄確認をチェックリスト化した。
