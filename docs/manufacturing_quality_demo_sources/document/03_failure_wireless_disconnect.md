# 無線通信が断続的に切断される不具合

QS-100 の無線通信切断は、電源瞬断、アンテナ接続不良、ゲートウェイとの距離、周辺設備からのノイズ、ファームウェアの再送制御不具合に分けて調査する。顧客先では「日中だけ欠測する」「フォークリフトが通る時間帯にデータが抜ける」など、環境要因と混ざって報告される。

品質保証部は、欠測時刻、RSSI、電池電圧ログ、設置場所の写真、同一ゲートウェイ配下の他センサー状況を確認する。RSSI が安定しているのに再起動ログが残る場合は、電源基板や電池ホルダー接点の問題を疑う。RSSI が急落する場合はアンテナケーブルの半挿し、金属棚による遮蔽、ノイズ源の近接を確認する。

ファームウェア 1.1 系では再送キューが満杯になると送信復帰が遅れる既知不具合がある。現地交換前に、ファームウェア版数、設置距離、電池電圧、最終検査の無線感度結果を照合する。
