# 受入検査と部品ロット管理

受入検査では、センサーモジュール、無線モジュール、電源部品、樹脂ケースについて、外観、数量、梱包状態、主要電気特性を確認する。検査結果は部品ロット番号、納入日、サプライヤー、検査担当者、判定結果とともに品質記録へ登録する。

重要部品は抜取基準を通常検査と強化検査に分ける。新規サプライヤー、工程変更後、過去 3 か月以内に不具合を出した部品は強化検査とし、測定点数を増やす。部品ロット A23 のコンデンサ問題後は、電源基板用コンデンサに対して熱衝撃後の容量測定と端面クラック確認を追加した。

受入検査で不合格になった部品は、隔離棚に移動し、購買、製造、品質保証へ通知する。特採で使用する場合は、使用製品、対象数量、追加検査、顧客影響を品質保証責任者が承認する必要がある。
