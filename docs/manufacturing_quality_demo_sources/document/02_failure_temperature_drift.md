# 温度測定値が高めにずれる不具合

温度測定値が実温より 1.5 度から 3 度ほど高く表示される場合、センサーモジュールの個体差、校正係数の登録漏れ、ケース内部の発熱、設置環境の直射日光が主な確認対象になる。顧客からは「室温計より常に高い」「夜間は合うが日中だけ高い」と報告されることが多い。

調査では、対象シリアル番号から製造ロット、校正ログ、ファームウェア版数を確認する。校正ログが存在しない場合は、個体校正工程で治具通信エラーが発生し、暫定係数のまま出荷された可能性がある。ファームウェア 1.2.0 では温度補正テーブルの丸め処理に問題があり、高温側で値が上振れする事例があった。

再現確認では恒温槽で 10 度、25 度、40 度の三点測定を行い、同一ロット品と良品基準品を比較する。是正処置として校正ログ未登録品を出荷停止し、最終検査で校正係数の空欄チェックを追加した。
