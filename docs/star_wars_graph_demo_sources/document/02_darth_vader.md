## タイトル
ダース・ベイダーの人物記録

## 本文
ダース・ベイダーは銀河帝国の恐怖政治を体現した人物であり、その正体はアナキン・スカイウォーカーである。アナキンはもともと惑星タトゥイーン出身で、幼少期には奴隷として暮らしていたが、クワイ＝ガン・ジンとオビ＝ワン・ケノービに見いだされてジェダイ・オーダーへ導かれた。ジェダイとしてのアナキンは高い操縦技術と戦闘能力を持ち、クローン戦争では共和国側の英雄として名を上げた。しかし、ヨーダやジェダイ評議会との距離、愛する者を失うことへの恐れ、そしてパルパティーンによる誘惑が重なり、アナキンはシスの弟子ダース・ベイダーへと変貌した。ムスタファーでオビ＝ワンと決裂した後、彼は生命維持装置付きの黒い装甲をまとい、銀河帝国の執行者となる。

帝国期のダース・ベイダーは、皇帝パルパティーンの最重要配下として帝国軍全体に影響を与えた。彼は帝国軍の提督や総督より上位に近い政治的圧力を持ち、ウィルハフ・ターキンや各地の指揮官と協力しながら、反乱同盟軍と生き残りのジェダイを追跡した。デス・スター計画の運用にも関与し、レイア・オーガナから反乱同盟軍の情報を引き出そうとしたほか、タトゥイーンやデス・スター内部ではオビ＝ワン・ケノービと再会している。宇宙戦ではTIEアドバンストx1を操縦し、ホスの戦いではスーパー・スター・デストロイヤーであるエグゼキューターから侵攻を統率した。ベスピンではランド・カルリジアンの行政権を利用してハン・ソロ一行を罠にかけ、ルーク・スカイウォーカーを自陣へ引き込もうとした。

ダース・ベイダーという存在の重要性は、単なる帝国軍司令官ではなく、親子関係と敵対関係が同時に成立する点にある。彼はルーク・スカイウォーカーとレイア・オーガナの父でありながら、その事実を長く秘匿し、反乱同盟軍の敵として行動した。ジェダイ・オーダーに対しては裏切り者であり、ヨーダやオビ＝ワンの弟子系譜に対する破壊者でもあった。一方でエンドアの戦いでは、第二デス・スターにおけるルークとの対話を通じて、シスの弟子としての立場より父としての良心が勝る。最終局面でダース・ベイダーはパルパティーンに反旗を翻し、ルークを救うため皇帝を排除した。この行為により、アナキン・スカイウォーカーとしての側面が回復し、銀河帝国の崩壊に直結する出来事を引き起こした。登場エピソードはエピソード3、エピソード4、エピソード5、エピソード6が中核となる。
