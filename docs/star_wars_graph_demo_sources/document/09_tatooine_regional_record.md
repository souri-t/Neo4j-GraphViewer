## タイトル
タトゥイーンの地域記録

## 本文
タトゥイーンは外縁部に位置する砂漠惑星であり、銀河中心の政治や軍事から距離を置きながらも、スカイウォーカー家、ジャバ・ザ・ハット、オビ＝ワン・ケノービ、ルーク・スカイウォーカー、ダース・ベイダーの起点となった重要地域である。乾燥地帯にはラーズ家の湿気農場が点在し、交易港モス・アイズリーには密輸業者、傭兵、ドロイド商人、帝国軍の監視要員が出入りする。さらに外周ではサンド・ピープルとして知られるタスケン・レイダーが活動し、ハット・カルテルはジャバの宮殿を中心に独自の支配圏を築いていた。辺境であるがゆえに統治の空白が多く、同時に多様な人物と組織が交差しやすいのがタトゥイーンの特徴である。

この惑星はアナキン・スカイウォーカーとルーク・スカイウォーカーという二世代の起点である。アナキンは奴隷身分から離脱してジェダイ・オーダーへ入り、その後ダース・ベイダーとなって帝国軍の執行者になった。一方、ルークはオーウェン・ラーズとベルー・ラーズのもとで育ち、農場生活から反乱同盟軍の英雄へ変化する。オビ＝ワン・ケノービはムスタファーでアナキンと決裂した後、この惑星へ潜伏し、遠くからルークを保護した。また、レイア・オーガナのメッセージを受けたR2-D2とC-3POがラーズ家に到着した出来事は、タトゥイーンをデス・スター攻略戦の出発点に変えている。モス・アイズリーではハン・ソロとチューバッカ、ミレニアム・ファルコンとの邂逅も起きた。

タトゥイーンは銀河内戦後期にも重要であり、ハン・ソロ救出作戦やジャバ・ザ・ハットとの対立を通じて、反乱同盟軍の非正規作戦とも接続する。帝国軍、ハット・カルテル、地方住民、ジェダイの潜伏者、密輸業者が重なるため、この惑星を中心に人物、組織、宇宙船、ドロイド、出来事を多数結び付けられる。タトゥイーン関連の記録は、登場エピソードがエピソード1、エピソード2、エピソード4、エピソード6にまたがり、スカイウォーカー家の関係史やオビ＝ワン・ケノービの人物記録とも強く重なる。特に「出身地」と「再出発の地」が同じ惑星に重なるため、家族史と戦史の両面から再参照しやすい。グラフDBでは「出身」「居住」「潜伏」「出会い」「救出」「敵対」といった多様なエッジを生成しやすい地域ノードである。
